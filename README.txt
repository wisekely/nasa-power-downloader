NASA Power Downloader - QGIS plugin 
--------------------------------------------------
Plugin title: NASA Power Downloader
Toolbar tooltip: Download all climatic data
Icon label: NASA Power Downloader

Installation:
1. In QGIS: Plugins > Manage and Install Plugins > Install from ZIP
2. Select NASA_Power_Downloader.zip and install
3. Restart QGIS if necessary
4. Enable the plugin and use toolbar button or Plugins menu

Notes:
- The plugin UI allows selecting multiple parameters and formats.
- Click 'Enable map click' to pick coordinates from the map canvas.
